Deskripsi Singkat Mata kuliah pemrograman web merupakan mata kuliah yang membahas tentang teknik pengembangan website dari hulu ke hilir, yaitu mulai dari algoritma dalam pemrograman web, pembuatan website (front-end dan back-end), hosting, hingga dokumentasi dan hukum kekayaan intelektual dari program yang dihasilkan nantinya.
Mata Kuliah
Materi Pembelajaran/ Pokok Bahasan Secara garis besar terdapat 2 materi yang akan diberikan yaitu:
1. Server Side Web Programming
Meliputi pemrograman web yang dilakukan pada bagian server dengan menggunakan bahasa pemograman Javascript.
2. Client Side Web Programming
Pemrograman web pada sisi client (browser) yang dilakukan dengan menggunakan bahasa pemrograman javascript.
